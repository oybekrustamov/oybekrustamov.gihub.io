Практическая работа #12.

Типы мобильных датчиков и работа с ними

Список Заданий

  1. Создать любое приложение и реализовать его с разными датчиками телефона
  2. Сделать так, чтобы по повороту телефона выводился звук через динамики о состоянии телефона
  3. Сделать программу, которая сообщает о заряде в 70-80-90-100% с помощью звука
  4. Сделать программу, которая по снижению заряда уменьшал и яркость экрана
