Практическая работа #15.

Разработка iOS приложений

Список Заданий

  1. Изучить архитектуру iOS
  2. Изучить язык Swift
  3. Изучить язык Objective-C
  4. Ознакомиться с элементами UI iOS.
  5. Создать любое простое приложение для iOS

Для того, чтобы разрабатывать приложение под iOS необходимо иметь компьютер с операционной системой MacOS и среду разработки XCODE
